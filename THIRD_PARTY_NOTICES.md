The full-screen reading stage, compact reading chrome, bottom navigation,
bottom-sheet controls, responsive paginated/scrolling presentation, and related
interaction patterns in this repository were adapted for the existing Vue 3,
Naive UI, and IndexedDB architecture from Candle Reader.

prebuilt bundles, Vuetify application shell, epub.js runtime, or remote-service
integration. EPUB parsing, translation alignment, local data migration, and
reader storage remain implementations of this project.
